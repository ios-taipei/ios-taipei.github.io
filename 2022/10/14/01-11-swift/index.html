    Range 範圍

    封閉式 Closed Range

    for number in 1...5 {
  print(number)
}

    range 只有正向的寫法，沒有逆向顛倒的寫法

    以下寫法是有問題的

    for number in 5...1 {
  print(number)
}

    半開放 Half-Open Range

    for number in 1..<4 {
  print(number)
}

    單邊 One-Sided Range

    let range = ..5
print(range.contains(-1))
